# OCR Перекладач EN → UA

Переклад тексту з екрану без встановлення Python.

---

## ЯК ОТРИМАТИ .EXE ФАЙЛ (покрокова інструкція)

### Крок 1 — Створіть безкоштовний акаунт на GitHub
Перейдіть на https://github.com і зареєструйтесь.

### Крок 2 — Створіть новий репозиторій
1. Натисніть кнопку "+" у правому верхньому куті → "New repository"
2. Назва: ocr-translator
3. Виберіть "Public"
4. Натисніть "Create repository"

### Крок 3 — Завантажте файли
Натисніть "uploading an existing file" і завантажте ВСІ ці файли:
  - ocr_translator.py
  - icon.ico
  - .github/workflows/build.yml  ← важливо зберегти цей шлях!

Для файлу build.yml:
1. Натисніть "Create new file"
2. В полі імені введіть:  .github/workflows/build.yml
3. Вставте вміст файлу build.yml
4. Натисніть "Commit changes"

### Крок 4 — Запустіть збірку
1. Перейдіть на вкладку "Actions" у вашому репозиторії
2. Ліворуч виберіть "Build Windows EXE"
3. Натисніть "Run workflow" → "Run workflow"
4. Зачекайте 3-5 хвилин (зелена галочка = готово)

### Крок 5 — Скачайте .EXE
1. Клікніть на завершений запуск (зелена галочка)
2. Внизу сторінки знайдіть розділ "Artifacts"
3. Натисніть "OCR_Translator_Windows" — завантажиться ZIP
4. Розпакуйте ZIP — всередині OCR_Translator.exe

---

## ЯК КОРИСТУВАТИСЬ ПРОГРАМОЮ

Перед першим запуском — встановіть Tesseract OCR (безкоштовно):
  https://github.com/UB-Mannheim/tesseract/wiki
  → завантажте tesseract-ocr-w64-setup-*.exe
  → встановіть (мову "English" вибрано за замовчуванням)

Запуск: двічі клікніть OCR_Translator.exe

Використання:
  1. Натисніть "📷 Зробити скріншот" або Ctrl+Shift+S
  2. Виділіть мишею область з англійським текстом
  3. Отримайте переклад українською

Примітка: потрібен інтернет для перекладу.
